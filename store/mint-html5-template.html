Mint Mobile App HTML5 Mobile friendly Template. Specially design for Mobile App Developers to Showcase the app to the Customers. If you do the UI Design, Mobile App Development, App Marketing, and you want to show your Mobile app to users this template will help you. This template is parallax model. It will occupy the full width of the browser window.

This template will compatible for all browsers both older versions and newer modern browsers. And also for all kinds of device table, mobile, desktop, and laptop.

This template will easy to manage and easy to customize. This template has the About Us, Services, Template, Screen Shot and Contact Section.
